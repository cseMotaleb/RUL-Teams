A WordPress plugin for managing team members with full CRUD functionality, sorting, bulk actions, and AJAX support.

== Description ==

The RUL Teams plugin allows you to manage your team members effortlessly. It provides a user-friendly admin interface to add, edit, delete, and view team members. Key features include:

- **CRUD Operations**: Add, edit, view, and delete team members.
- **AJAX Support**: Perform actions like single delete and bulk delete without reloading the page.
- **Bulk Actions**: Delete multiple team members at once.
- **Pagination and Search**: Navigate through team members and search them by name.
- **Sorting**: Sort team members by name or other attributes.
- **WordPress Standards**: Built using WordPress coding standards and object-oriented principles.

== Installation ==

1. Download the plugin.
2. Upload the `rul-teams` folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Navigate to **RUL Teams** in the admin dashboard to manage team members.

== Features ==

- User-friendly admin interface.
- Add, edit, and delete team members.
- Sort team members by name.
- AJAX-powered single and bulk delete actions.
- Pagination and search options for easy navigation.
- Extendable and developer-friendly codebase.

== Usage ==

1. Navigate to **RUL Teams** in your WordPress admin dashboard.
2. Use the "Add New" button to add a team member.
3. Use the edit option to update team member details.
4. Perform single or bulk delete actions with AJAX support.
5. Search for team members using the search bar at the top.

== Frequently Asked Questions ==

= How do I add a team member? =
Navigate to **RUL Teams > Add New**, fill out the form, and click "Add Team Member."

= Can I sort team members? =
Yes, team members can be sorted by their names or other attributes.

= Does this plugin support AJAX? =
Yes, AJAX is used for single delete, bulk delete, and seamless updates.

= What happens if I deactivate the plugin? =
The plugin will stop functioning, but no data will be deleted.
